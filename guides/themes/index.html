Themes

Themes allow you to change the look and feel of the generated documentation. You can use one of the included themes, modify them to suit your needs or create a fully custom theme. The --theme parameter allows you to select a theme when creating a documentation:

TypeDoc ships with one builtin default theme, additional themes can be added by plugins. For documentation on creating custom themes, see the custom-themes.md document in the TypeDoc repository.

You can find more themes on npm under the typedoc-theme keyword.
